Macro Variables Avec SAS

Dernière mise à jour le Jul 1, 2021 1 min de lecture SAS

Les macro-variables dans SAS permettent de substituer du texte dans un programme SAS. Ils facilitent ainsi la programmation en évitant la répétition du code et le copier-coller. Ils facilent aussi la maintenance des programmes SAS ainsi que la lecture. Il existe trois façons de créer des macro-variables dans un programme SAS:

  1. Dans un code ouvert avec l'instruction %let
  2. Avec la routine call symputx
  3. Dans une procédure PROC SQL 
proc sql noprint;
select distinct make into :cars from sashelp.cars;
bibi.data;
quit;
/****************/
proc print data=sashelp.cars;
var test;
title 'bonjour';
run;
***********************************************************************************;
*                          MACRO PROGRAMME                                        *;
***********************************************************************************;
%macro test;
%do i=1 %to 10;
proc print data= sashelp.cars;
var make&i;
run;
%mend;
%test

`
